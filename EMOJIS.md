# 🎨 Emojis für BotForge

Hier sind die offiziellen Emojis, die du auf deinem Server hochladen solltest, damit BotForge sie nativ in Embeds oder Dropdowns verwenden kann. 

Da Discord benutzerdefinierte Emojis über eine ID lädt (`<:name:id>`), ist es am besten, ein Set von sauberen Emojis für deinen Server zu definieren.

## Empfohlene Emojis zum Herunterladen
Du kannst diese Symbole als `.png` speichern und in die Server-Einstellungen unter **Emojis** hochladen.

1. **Ticket-Kategorien**
   - 🎫 `ticket_general`
   - ⚠️ `ticket_report`
   - 🤝 `ticket_partner`
   - 🐛 `ticket_bug`

2. **Moderation**
   - 🔨 `mod_ban`
   - 👢 `mod_kick`
   - ⏱️ `mod_timeout`
   - ⚠️ `mod_warn`

3. **Status**
   - 🟢 `status_online`
   - 🟡 `status_idle`
   - 🔴 `status_dnd`
   - ⚪ `status_offline`

4. **Bot-Aktionen**
   - ✅ `success`
   - ❌ `error`
   - ℹ️ `info`
   - ⚙️ `settings`

> **Tipp:** Wenn du die Emojis hochgeladen hast, kannst du im Dashboard unter **Custom Emojis** den Namen des Emojis angeben, und der Bot verwendet dieses automatisch!
